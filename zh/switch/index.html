# Switch 开关选择器

开关选择器。

# 何时使用

  • 需要表示开关状态/两种状态之间的切换时；
  • 和 checkbox的区别是，切换 switch 会直接触发状态改变，而 checkbox 一般用于状态标记，需要和提交操作配合。

# 代码演示

# 基本

最简单的用法。

<Vue source code> 
<template>
  <section>
    <NebulaSwitch defaultChecked @change="onChange"/>
  </section>
</template>

<script>
export default {
  methods: {
    onChange(checked) {
      console.log(`switch to ${checked}`);
    }
  }
};
</script>

# 不可用

Switch 失效状态。

<Vue source code> 
<template>
  <main>
    <section>
      <NebulaSwitch :disabled="disabled" defaultChecked />
    </section>
    <section>
      <NebulaButton type="primary" @click="toggle">Toggle disabled</NebulaButton>
    </section>
  </main>
</template>

<script>
export default {
  data() {
    return {
      disabled: true
    };
  },
  methods: {
    toggle() {
      this.disabled = !this.disabled;
    }
  }
};
</script>

# 文字和图标

带有文字和图标。

开
0
<Vue source code> 
<template>
  <main>
    <section>
      <NebulaSwitch checkedChildren="开" unCheckedChildren="关" defaultChecked/>
    </section>
    <section>
      <NebulaSwitch checkedChildren="1" unCheckedChildren="0"/>
    </section>
    <section>
      <NebulaSwitch defaultChecked>
        <NebulaIcon type="check" slot="checkedChildren"/>
        <NebulaIcon type="cross" slot="unCheckedChildren"/>
      </NebulaSwitch>
    </section>
  </main>
</template>

# 两种大小

size="small" 表示小号开关。

<Vue source code> 
<template>
  <main>
    <section>
      <NebulaSwitch defaultChecked/>
    </section>
    <section>
      <NebulaSwitch size="small" defaultChecked/>
    </section>
  </main>
</template>

# 加载中

标识开关操作仍在执行中。

<Vue source code> 
<template>
  <main>
    <section>
      <NebulaSwitch loading defaultChecked/>
    </section>
    <section>
      <NebulaSwitch size="small" loading/>
    </section>
  </main>
</template>

# API

参数 说明 类型 默认值
autoFocus 组件自动获取焦点 boolean false
checked 指定当前是否选中 boolean false
checkedChildren 选中时的内容 string|VueNode
defaultChecked 初始是否选中 boolean false
disabled 是否禁用 boolean false
loading 加载中的开关 boolean false
size 开关大小，可选值：default small string default
unCheckedChildren 非选中时的内容 string|VueNode
onChange 变化时回调函数 Function(checked:Boolean)

# 方法

名称 描述
blur() 移除焦点
focus() 获取焦点
