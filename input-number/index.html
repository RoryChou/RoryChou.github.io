# InputNumber

Enter a number within certain range with the mouse or keyboard.

# When To Use

When a numeric value needs to be provided.

# Examples

# Basic

Numeric-only input box.

<Vue source code> 
<template>
  <section>
    <NebulaInputNumber :min="1" :max="10" :defaultValue="3" @change="onChange"/>
  </section>
</template>

<script>
export default {
  methods: {
    onChange(value) {
      console.log("changed", value);
    }
  }
};
</script>

# Sizes

There are three sizes available to a numeric input box. By default, the size is 30px. The two additional sizes are large and small which means 40px and 24px, respectively.

<Vue source code> 
<template>
  <section class="demo-input-number-size">
    <NebulaInputNumber size="large" :min="1" :max="100000" :defaultValue="3" @change="onChange"/>
    <NebulaInputNumber :min="1" :max="100000" :defaultValue="3" @change="onChange"/>
    <NebulaInputNumber size="small" :min="1" :max="100000" :defaultValue="3" @change="onChange"/>
  </section>
</template>

<script>
export default {
  methods: {
    onChange(value) {
      console.log("changed", value);
    }
  }
};
</script>

<style lang="less" scoped>
.demo-input-number-size {
  .nebula-input-number {
    margin-right: 10px;
  }
}
</style>

# Disabled

Click the button to toggle between available and disabled states.

<Vue source code> 
<template>
  <main>
    <section>
      <NebulaInputNumber :min="1" :max="10" :disabled="disabled" :defaultValue="3"/>
    </section>
    <section>
      <NebulaButton @click="toggle" type="primary">Toggle disabled</NebulaButton>
    </section>
  </main>
</template>

<script>
export default {
  data() {
    return {
      disabled: true
    };
  },
  methods: {
    toggle() {
      this.disabled = !this.disabled;
    }
  }
};
</script>

# Decimals

A numeric-only input box whose values can be increased or decreased using a decimal step. The number of decimals (also known as precision) is determined by the step prop.

<Vue source code> 
<template>
  <section>
    <NebulaInputNumber :min="0" :max="10" :step="0.1" @change="onChange"/>
  </section>
</template>

<script>
export default {
  methods: {
    onChange(value) {
      console.log("changed", value);
    }
  }
};
</script>

# Formatter

Display value within it's situation with formatter, and we usually use parser at the same time.

<Vue source code> 
<template>
  <main>
    <section>
      <NebulaInputNumber
        :defaultValue="1000"
        :formatter="value => `$ ${value}`.replace(/\B(?=(\d{3})+(?!\d))/g, ',')"
        :parser="value => value.replace(/\$\s?|(,*)/g, '')"
        @change="onChange"
      />
    </section>
    <section>
      <NebulaInputNumber
        :defaultValue="100"
        :min="0"
        :max="100"
        :formatter="value => `${value}%`"
        :parser="value => value.replace('%', '')"
        @change="onChange"
      />
    </section>
  </main>
</template>

<script>
export default {
  methods: {
    onChange(value) {
      console.log("changed", value);
    }
  }
};
</script>

# API

property description type default
autoFocus get focus when component mounted boolean false
defaultValue initial value number
disabled disable the input boolean false
formatter Specifies the format of the value presented function(value: number | string): string -
max max value number Infinity
min min value number -Infinity
parser Specifies the value extracted from formatter function( string): number -
precision precision of input value number -
decimalSeparator decimal separator string -
size width of input box string -
step The number to which the current value is increased or decreased. It can be an integer or decimal. number|string 1
value current value number
onChange The callback triggered when the value is changed. function(value: number | string)
